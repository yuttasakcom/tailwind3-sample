# Tailwind CSS 3 Sample

## Install
```
npm install -D tailwindcss postcss autoprefixer
```

## Config
```
npx tailwindcss init
```
## Extension
```
- Tailwind CSS IntelliSense
- PostCSS Language Support
- Live Server
```

## Run Tailwind
```
npm run build
```

## Live Server
```
open live server -> index.html
```
